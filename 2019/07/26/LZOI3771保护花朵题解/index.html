3771: 保护花朵Protecting the Flowers

时间限制: 1 Sec 内存限制: 128 MB
提交: 84 解决: 33
[提交][状态][题解]
题目描述
有N (2 ≤ N ≤ 100,000) 头奶牛跑到FJ的花园里去吃花儿了，它们分别在距离牛圈T(1 ≤ Ti ≤ 2,000,000)分钟处吃花儿，每分钟会吃掉D (1 ≤ Di ≤ 100) 朵卡哇伊的花儿，（此处有点拗口，不要在意细节啊！），FJ现在要将它们给弄回牛圈，但是他每次只能弄一头回去，来回用时总共为2*T分钟，在这段时间内，其它的奶牛会继续吃FJ卡哇伊的花儿，速度保持不变，当然正在被赶回牛圈的奶牛就没口福了！现在要求以一种最棒的方法来尽可能的减少花儿的损失数量，求奶牛吃掉花儿的最少朵数！
输入
第1行:整数N

第2行..N+1行：两个整数，Ti和Di

输出
1行：奶牛吃掉花儿的最少朵数
样例输入
6
3 1
2 5
2 3
3 2
4 1
1 6
样例输出
86
提示

赶牛顺序：6, 2, 3, 4, 1, 5.

损失花朵：24 + 28 + 16 + 12 + 6 = 86.

    3. 3. 那么我们考虑两头牛x和y的先后顺序：
    4. 4. 假设我们先赶走牛x再赶走牛y，那么我们的损失应该是：
    5. 5. \sum_{i 包括y的未被赶走的牛}(2*D[i]*T[x])
    6. 6. 而如果先赶走牛$y$再赶走牛$x$的话，我们的损失类似的应该是：
    7. 7. \sum_{i 包括x的未被赶走的牛}(2*D[i]*T[y]))
    8. 8. 按照Cmp的规律，先选x的损失应比y要小，即：
    9. 9. \sum_{i 包括y的未被赶走的牛}(2*D[i]*T[x]) < \sum_{i 包括x的未被赶走的牛}(2*D[i]*T[y])
    10. 10. 将左边的y和右边的x分别单独提出来，顺便把T[x]和T[y]提出来
    11. 11. 2*D[y]*T[x]+\sum_{i 未被赶走的牛}(2*D[i])*T[x] < 2*D[x]*T[y]+\sum_{i 未被赶走的牛}(2*D[i])*T[y]
    12. 12. 同时除以2
    13. 13. D[y]*T[x]+\sum_{i 未被赶走的牛}D[i]*T[x] < D[x]*T[y]+\sum_{i 未被赶走的牛}D[i]*T[y]
    14. 14. 设\sum_{i 未被赶走的牛}D[i]=ss
    15. 15. 那么原式变成： D[y]*T[x]+ss*T[x] < D[x]*T[y]+ss*T[y]
    16. 16. 然后可以直接将ss*T这一项去掉
    17. 17. 那么：D[y]*T[x] < D[x]*T[y]
    18. 18. 那就完了呀，直接按照这个排序，然后慢慢求解即可

      2. 18.2. 对于牛x，赶x的损失当然就是\sum_{i=x+1}^n D[i] *2*T[x]
    19. 19. 其中x+1~n就是还没赶走的牛。为什么呢？因为我们都排完序了，是按1~n的顺序赶的
